# INSAIT Security Research Group Website

Static website for the INSAIT Security Research Group, featuring research topics, team members, publications, and contact information.

## Structure

```
docs/                    → Published static site (HTML/CSS/JS)
├── index.html           → Home page
├── research.html        → Research topics
├── members.html         → Team members
├── publications.html    → Publications
├── contact.html         → Contact page
├── static/
│   ├── css/style.css    → Custom styles
│   └── js/main.js       → Custom JavaScript
└── media/members/       → Member photos

members/                 → Source member photos
```

## Development

Edit files in `docs/` directly. Open any `.html` file in a browser to preview.
